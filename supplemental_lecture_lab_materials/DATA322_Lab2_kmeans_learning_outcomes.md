# DATA 322 Lab 2: K-Means Learning Outcomes (Draft)

## Learning Outcomes

By the end of Lab 2, students should be able to:

1. Explain the k-means objective and how iterative centroid updates produce cluster assignments.
2. Fit k-means models for different values of `k` and compare resulting partitions.
3. Evaluate cluster quality with a metric such as within-cluster variation.
4. Interpret cluster centroids in the context of original variables.
5. Describe how scaling and feature selection might affect distance-based clustering outcomes.
