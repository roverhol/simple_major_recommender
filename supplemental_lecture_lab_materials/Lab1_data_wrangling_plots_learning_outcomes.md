# Lab 1: Data wrangling and plots — learning outcomes
## Learning Outcomes

By the end of Lab 1, students should be able to:

1. Import and inspect a tabular dataset, identify key variable types and check for missing data.
2. Perform key data wrangling operations such as filtering, selecting, creating new variables from old variable variables, and grouping.
3. Reshape or aggregate data into an appropriate structure for downstream modeling tasks.
4. Produce simple visualizations of one to three variables
